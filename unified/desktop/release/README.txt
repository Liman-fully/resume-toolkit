简历整理工具 v1.0.0 - 安装说明
=====================================

📦 文件说明
-----------
简历整理工具-1.0.0.dmg (145 MB)
- macOS 桌面应用程序安装包
- 支持 macOS 10.13 及以上版本
- 支持 Intel 和 Apple Silicon 芯片

🚀 安装步骤
-----------
1. 双击打开 "简历整理工具-1.0.0.dmg"
2. 将 "简历整理工具" 拖拽到 Applications 文件夹
3. 在 Launchpad 或 Applications 中找到应用
4. 双击运行

⚠️  首次运行注意事项
--------------------
如果系统提示"无法打开，因为无法验证开发者"：
1. 前往 系统设置 > 隐私与安全性
2. 在"安全性"部分点击"仍要打开"
3. 或者右键点击应用图标，选择"打开"

📋 使用说明
-----------
1. 启动应用后，选择"源目录"（包含简历的文件夹）
2. 选择"目标目录"（整理后的简历存放位置）
3. 点击"预览"查看整理效果
4. 确认无误后，取消"预览模式"，点击"执行"

🔒 安全说明
-----------
- 这是一个桌面应用程序，所有操作在本地完成
- 不需要网络连接
- 不会上传任何数据到云端
- 默认预览模式，不会实际移动文件
