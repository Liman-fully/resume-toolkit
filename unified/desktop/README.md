# 简历整理工具 - 桌面版 v2.0

一个真正的桌面应用，双击即可运行，无需安装 Python 环境。

## 特性

- 🚀 **双击运行**：无需 Python 环境，独立可执行文件
- 🔒 **代码保护**：核心逻辑混淆打包，防止逆向
- 📂 **原生权限**：直接访问文件系统，支持深度扫描
- 💻 **跨平台**：Windows / macOS / Linux 三个版本
- 🎨 **现代 UI**：Vue 3 + Vite，流畅的交互体验
- ⚡ **实时进度**：显示处理进度，让用户清楚执行过程

## 安装与使用

### 开发模式

```bash
npm install
npm run dev
```

### 打包发布

```bash
# 打包所有平台
npm run build

# 只打包 Windows
npm run build:win

# 只打包 macOS
npm run build:mac

# 只打包 Linux
npm run build:linux
```

打包后的可执行文件位于 `release/` 目录：
- Windows: `resume-toolkit-desktop Setup 1.0.0.exe`
- macOS: `简历整理工具-1.0.0.dmg`
- Linux: `简历整理工具-1.0.0.AppImage`

## 架构设计

### 三层架构

```
┌─────────────────────────────────────┐
│   Renderer (Vue 3)                  │
│   - UI 界面                          │
│   - 只负责显示，无业务逻辑           │
│   - 通过 preload 与主进程通信        │
└──────────────┬──────────────────────┘
               │ IPC (安全沙箱)
┌──────────────▼──────────────────────┐
│   Preload (桥接层)                   │
│   - 暴露安全的 API                   │
│   - contextIsolation: true          │
│   - nodeIntegration: false          │
└──────────────┬──────────────────────┘
               │ IPC
┌──────────────▼──────────────────────┐
│   Main (Node.js)                    │
│   - 核心逻辑（混淆）                 │
│   - 文件系统操作                     │
│   - 简历解析/去重/归类               │
└─────────────────────────────────────┘
```

### 安全保护措施

| 保护方式 | 说明 |
|---------|------|
| **代码混淆** | 核心算法使用 JavaScript Obfuscator 混淆 |
| **Asar 打包** | 源码打包成 `.asar` 文件，不易读取 |
| **逻辑分离** | 核心逻辑在 Node.js，前端只调用 API |
| **沙箱隔离** | Preload 启用上下文隔离，禁用 node 集成 |
| **最小权限** | 只暴露必要的 API 给前端 |

### 与 Web UI 的对比

| 特性 | Web UI | Electron 桌面版 |
|------|--------|----------------|
| 文件系统权限 | ❌ 需要后端 | ✅ 原生支持 |
| 用户安装 | ❌ 需要配置服务器 | ✅ 双击即可 |
| 代码保护 | ❌ 源码暴露 | ✅ 混淆打包 |
| 用户体验 | ❌ 浏览器限制 | ✅ 原生窗口 |
| 跨平台 | ✅ 任何浏览器 | ✅ 三个版本 |

## 构建流程

1. **开发**：`npm run dev`
2. **混淆**：`node scripts/obfuscate.js`（混淆核心逻辑）
3. **构建**：`npm run build`（生成可执行文件）
4. **发布**：将 `release/` 目录下的文件分发

## 用户使用

普通用户只需：
1. 下载对应平台的可执行文件
2. 双击运行
3. 选择源目录和目标目录
4. 点击预览/执行

无需任何技术背景！
