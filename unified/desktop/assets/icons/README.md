# 应用图标设计

## 设计理念

1. **核心元素**：
   - 📄 简历/文档符号
   - 🔄 整理/分类动作
   - ✨ 效率/速度

2. **配色方案**：
   - 主色：蓝色 #2196F3（专业、信任）
   - 辅色：橙色 #FF9800（活力、创新）
   - 背景：白色/渐变

3. **风格**：
   - 扁平化设计
   - 圆角矩形
   - 简洁现代

## 尺寸要求

### Windows (.ico)
- 16x16
- 32x32
- 48x48
- 256x256

### macOS (.icns)
- 16x16
- 32x32
- 64x64
- 128x128
- 256x256
- 512x512
- 1024x1024

### Linux (.png)
- 128x128
- 256x256
- 512x512

## 设计概念

### 方案 A：文档整理
- 中心：一张堆叠的简历
- 动作：箭头表示整理方向
- 背景：柔和渐变

### 方案 B：AI 助手
- 中心：机器人图标 + 文档
- 动作：机器人整理文档
- 风格：科技感

### 方案 C：抽象符号
- 中心：抽象的文件夹 + 文档
- 动作：简洁线条
- 风格：极简主义
